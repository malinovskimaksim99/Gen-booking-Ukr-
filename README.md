# Gen-booking-Ukr
Локальний AI-письменник (українська модель) з деревом сюжету та навчанням QLoRA.

## Швидкий старт (Mac)
1. Встанови Python 3.11+  
   brew install python
2. У папці проєкту:
   pip install -r requirements.txt
   ./scripts/setup_mac.sh
3. Запуск:
   ./scripts/start_dev.sh
4. Відкрий http://localhost:5173
   Пароль за замовчуванням: qwerty123
